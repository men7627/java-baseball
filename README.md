  |*Menu*|**printMenu()**|메뉴 화면 출력 (1.게임 시작, 2.게임 종료)|O|

  |*Number*|**getFirst()**|첫번째 숫자 읽어옴|O|

  |*Number*|**setSecond()**|두번째 숫자 지정|O|
